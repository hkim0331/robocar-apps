# Robocar-Apps

バラバラに書いていた情報処理応用（ロボカー）授業用の web アプリ をまとめる。

* コードの重複を避ける
* hunchentoot の複数立ち上げを抑制する
* vm2016 のポート使用を抑制する
* web アプリケーションごとに nginx を設定する煩わしさを軽減する

## start

M-x slime
CL-USER> (ql:quickload :robocar-apps)
